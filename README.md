# Customer-Behaviour-Dashboard
This project analyzes customer behavior patterns using PostgreSQL, Python (Pandas), and Power BI to derive actionable business insights. The goal is to understand customer demographics, purchasing behavior, and engagement trends, and present them through an interactive dashboard for decision-making.

The project follows an end-to-end analytics workflow:
-Raw data ingestion
-Data cleaning & exploratory analysis
-SQL-based analysis
-Interactive visualization

Tech Stack
-Database: PostgreSQL
-Programming: Python (Pandas)
-Visualization: Power BI
-Data Format: CSV
-Tools: pgAdmin, Jupyter Notebook
